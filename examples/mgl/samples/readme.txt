
                        ***********************
                        * MGL Sample Programs *
                        ***********************

Supported environments:
    32 bit DOS protected mode
    32 bit Windows

Color depths supported:
    All.

Interface:
    Command line for DOS, Dialog Box for Windows.

Description:
    This directory contains a number of small sample programs showing
    how to use specific parts of the MGL API. We dont provide IDE files
    for all the samples in this directory, however we do provide an
    IDE file for the MGLDEMO sample program, which you can easily convert
    to IDE files to build any of the remaining samples.
