
                  **************************************
                  * MGL Game Framework Sample Programs *
                  **************************************

Supported environments:
    32 bit DOS protected mode
    32 bit Windows

Color depths supported:
    All.

Interface:
    Command line for DOS.
    Font end dialog box for Windows.

Description:
    This directory contains sample programs for the MGL Game Framework,
    which is a small library with source code showing how to build
    commercial quality games using the MGL. There are two sample
    programs in here:

    BOUNCE - Simple demo showing a bouncing ball.

	GEARS  - Simple OpenGL demo showing spinning gears.

	SPRITE - Simple demo showing how to use the sprite manager.
