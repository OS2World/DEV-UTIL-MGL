MGL_unloadCursor

Unloads a cursor file from memory.

Declaration

void MGLAPI MGL_unloadCursor(
    cursor_t *cursor)

Prototype In

mgraph.h

Parameters

cursor

Pointer to cursor to unload

 

Description

Unloads the specified cursor file from memory, and frees up all the system resources associated with this cursor.
