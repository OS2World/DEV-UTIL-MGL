MGL_pixel

Draws a pixel at the specified location.

Declaration

void MGL_pixel(
    point_t p)

Prototype In

mgraph.h

Parameters

p

Point to plot pixel at

 

Description

This function is the same as MGL_pixelFast, however it takes the coordinate of the pixel to plot as a point.
